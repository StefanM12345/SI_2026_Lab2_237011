## Cyclomatic Complexity - searchBookByTitle

Цикломатската комплексност е пресметана според формулата:

M = P + 1

Во функцијата searchBookByTitle има:
- 3 if услови
- 1 for циклус

Затоа:

M = 4 + 1 = 5

Цикломатската комплексност на функцијата е 5.
